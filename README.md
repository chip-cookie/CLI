# Jeongong Bloom (정공블룸)

**Vibe Coding CLI** - AI 기반 설계 에이전트로, 웹 AI 빌더용 "AI-Ready Context Package"를 생성합니다.

## 기능

- **계획 수립**: 추상적인 아이디어를 구체적인 실행 단계로 분해합니다.
- **아키텍처 설계**: 데이터베이스 스키마, API 라우트, 핵심 로직을 생성합니다.
- **리팩토링**: AI가 이해하기 쉬운 형태로 코드를 최적화합니다.
- **컨텍스트 패키징**: v0, Bolt, ChatGPT 등에 최적화된 프롬프트를 출력합니다.

## 빠른 시작

```bash
# 의존성 설치
pip install -r requirements.txt

# Jeongong Bloom 실행
python run_bloom.py
```

## 기반 프로젝트

이 프로젝트는 [OpenManus](https://github.com/FoundationAgents/OpenManus)를 기반으로 Vibe Coding 기능을 확장했습니다.
